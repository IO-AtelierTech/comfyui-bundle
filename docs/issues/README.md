# Issue Tracking

This directory tracks known issues, workarounds, and planned fixes.

## File Naming Convention

- `NNNN-short-description.md` - Issue files (numbered)
- Use lowercase, hyphens for spaces

## Issue Template

```markdown
# Issue Title

**Status**: open | investigating | resolved | wontfix
**Priority**: critical | high | medium | low
**Component**: mcp | docker | plugin:name | comfyui

## Description
Brief description of the issue.

## Steps to Reproduce
1. Step one
2. Step two

## Expected Behavior
What should happen.

## Actual Behavior
What actually happens.

## Workaround
Temporary solution if available.

## Resolution
How it was fixed (when resolved).
```
